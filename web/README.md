# UCreativity 前端项目

## 📖 项目简介
UCreativity 是一个面向个人创作者的**智能写作工作台**，当前版本聚焦「写作助手」功能。  
它帮助用户聚合 RSS 订阅源，通过 AI 生成写作灵感，并辅助完成文章创作。  
项目形态参考 GenSpark，初期为个人使用，后期可扩展为多用户平台。

---

## ✨ 核心功能
- **邮箱绑定**：接收功能更新、每日推送、选题推荐等消息。
- **RSS 订阅管理**：自定义订阅源、推送时间、推送数量、AI 翻译开关。
- **文章推送与预览**：每日定时推送订阅文章，支持历史存档。
- **AI 选题生成**：基于最新推送内容，生成多个写作选题。
- **文章创作**：选择选题后，可设定写作风格与目标人群，AI 生成 Markdown 文章，并提供在线编辑。

---

## 🛠 技术栈
- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **UI 组件库**：Ant Design / Semi Design（或自行封装）
- **状态管理**：Zustand / Redux Toolkit
- **样式**：TailwindCSS + CSS Modules
- **Markdown 编辑器**：bytemd / md-editor-rt
- **HTTP 请求**：axios
- **路由**：React Router v6
- **代码规范**：ESLint + Prettier + Husky

## 🚀 快速开始
1. **克隆仓库**
   ```bash
   git clone https://github.com/yourname/ucreativity.git
   cd ucreativity
   ```

2. **安装依赖**
   ```bash
   pnpm install
   ```

3. **配置环境变量**
   创建 `.env` 文件，参考 `.env.example` 添加后端 API 地址等。

4. **启动开发服务器**
   ```bash
   pnpm dev
   ```

5. **构建生产版本**
   ```bash
   pnpm build
   ```
